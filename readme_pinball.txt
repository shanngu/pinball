1.) Unzip the contents of Pinball.zip to the preferred folder.
2.) All necessary files, including the JAMA package, should be found in the src folder
3.) Pinball must be run from a console. For Windows, compile in the cmd box
	with the command -java *.java
4.) Run the Pinball class. Again, if you are using Windows, use the command java Pinball.java
5.) Alternatively, you can add the project to an IDE of your choice and run from there.
